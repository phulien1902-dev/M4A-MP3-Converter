# M4A → MP3 Converter Portable

Công cụ chuyển đổi hàng loạt file M4A sang MP3 dành cho giáo viên, người làm E-Learning và người dùng phổ thông.

## Tính năng

- Chuyển đổi hàng loạt M4A → MP3
- Chọn thư mục đầu vào và đầu ra
- Quét thư mục con
- Giữ nguyên chất lượng âm thanh
- Không cần Internet
- Hỗ trợ Windows 10/11

## Ứng dụng

- PowerPoint
- iSpring Suite
- ActivePresenter
- Camtasia
- CapCut
- Canva
- OBS Studio
